# 🚀 Backend com Node.js | DevClub

Este repositório contém meus estudos e projetos focados no desenvolvimento de servidores e APIs utilizando Node.js, conforme a trilha do DevClub.

## 🎯 Objetivos de Aprendizado:
* **Fundamentos de Node.js:** Entender o funcionamento do V8 e o Event Loop.
* **Gerenciamento de Pacotes:** Uso do `npm` e `yarn`.
* **Criação de APIs:** Desenvolvimento de rotas e controladores com **Express**.
* **Banco de Dados:** Integração com bancos (PostgreSQL/MongoDB) usando Docker e ORMs.
* **Middlewares:** Tratamento de erros, autenticação e segurança.

## 🛠️ Tecnologias e Ferramentas:
* Node.js
* Express
* Nodemon (Desenvolvimento)
* Insomnia / Postman (Testes de API)
