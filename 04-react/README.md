# ⚛️ Frontend com React.js | DevClub

Repositório dedicado ao estudo da biblioteca React para a criação de interfaces modernas, dinâmicas e de alta performance.

## 🎯 Objetivos de Aprendizado:
* **Componentização:** Criar interfaces reutilizáveis e organizadas.
* **Hooks do React:** Domínio de `useState`, `useEffect` e `useContext`.
* **Roteamento:** Navegação entre páginas com `react-router-dom`.
* **Consumo de APIs:** Integração do Frontend com o Backend usando **Axios**.
* **Estilização:** Uso de CSS-in-JS (Styled Components) para layouts modernos.

## 🛠️ Tecnologias e Ferramentas:
* React (Vite)
* Styled Components
* React Hooks
* Axios
